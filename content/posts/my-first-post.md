mis en place sur ce blog. Comment il fonctionne, quels sont les outils mis en œuvre

Mais si vous être curieux, n’hésitez surtout, fouillez, récupérez, copiez, collez…

## Comment ça marche ?

Ce site fonctionne grâce à Hugo, un générateur de site statique.

```
hugo server -D
```
